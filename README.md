# DecredJWallet
## Info
DecredJWallet is a GUI Wallet for Decred written in Java.

## Building
To build this application Intellij IDEA is recommended as well as Maven.
It requires JDK 1.8.

[Here is a guide to build it on Windows](WINDOWS-BUILD.md)
